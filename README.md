step-by-step process:
1. Clone the repository: git clone <repository-url>

2. Navigate to the repo directory: cd <repo-name>

3. Create a new branch to make your changes: git checkout -b feature-branch

4. Make your changes in the code (e.g., edit a file).

5. Stage the changes: git add <filename> or git add .

6. Commit the changes: git commit -m \describe your change"

7. Push the changes to the remote branch: git push origin feature-branch

8. Open a Pull Request on GitHub to merge your changes into the main branch.
